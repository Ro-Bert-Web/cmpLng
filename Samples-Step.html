Flr

The Flr function works very similarily to the Floor function that exists in robotC and every other programming language. The description is that it rounds the input down to the nearest whole number. It doesn't. For positive numbers it does, but for negative numbers, it actually rounds up. This Flr function fixes that problem. It rounds x down to the nearest whole number even if it is negative.

Cel

The Cel function fixes the same thing the Flr function did with Floor, but instead with Ceil. Ceil rounds up, but not for negative numbers, the Cel function fixes that. It rounds x up to the nearest whole number no matter its sign.

How to Use

To use either of these commands, you just input a value. They will then output the rounded value (Up or down dependent on which function);
